Key Files

Makefile
All common tasks: install, run, test, lint, docker, prod
Dockerfile
Python 3.12-slim image with Kedro installed. Runs kedro run by default.
Dockerfile.viz
Same base, runs kedro viz run on port 4141.
Dockerfile.airflow
Extends apache/airflow:2.10.5-python3.12. Installs Kedro and datasets. Used by the production stack.
docker-compose.yml
Development services: kedro runner, viz UI, docs nginx.
docker-compose.prod.yml
Production Airflow stack: postgres, redis, webserver, scheduler, triggerer, worker.
dags/wildbook_pipeline.py
Airflow DAG that runs kedro run --pipeline example on a @daily schedule.
src/wildbook_pipeline/pipelines/example/
Example pipeline with four nodes: load, preprocess, extract features, generate report.
